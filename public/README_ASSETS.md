# Assets Directory

Place your image files in this `public` folder:

- `logo.png` - Logo image for the header (recommended size: 200x50px or similar aspect ratio)

## Image Requirements

### logo.png
- Format: PNG (with transparency preferred)
- Recommended size: 200x50px or similar
- Should work on both light and dark backgrounds

## Note
If images are not provided, the application will use fallback text/colors.
